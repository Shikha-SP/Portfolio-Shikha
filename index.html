Projects


Hunting Reserve

My first Unity project, inspired by Nepal’s Dhorpatan Hunting Reserve. I created this game while learning Unity and Blender over 1.5 months, combining my passion for wildlife, conservation, and game development. Hunting Reserve is a first-person exploration and shooting game set in Nepal’s only legal hunting reserve. The game features a point-based scoring system, raycast shooting mechanics, and custom animations, providing an immersive gameplay experience while raising awareness about wildlife conservation.

Bank Management System

This is my first Java project. A simple Bank Management System built using Java and NetBeans IDE. It includes a basic GUI where users can: Deposit money Withdraw money Transfer between accounts Check balance I built this project while learning core Java concepts and how to design GUI applications using NetBeans' drag-and-drop interface. Tools Used: Java NetBeans IDE This project helped me understand object-oriented programming, GUI components, and event handling in Java.
